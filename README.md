## Run-time dependencies

Starting with v6.x, the minimum requirements for running the Eclipse
Embedded CDT plug-ins are:

- JavaSE-11 or later
- CDT 9.2.1 (Neon.3)

The recommended:

- JavaSE-11 or later
- Eclipse 2020-09 or later
